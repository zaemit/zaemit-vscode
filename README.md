# Zaemit Visual Editor

> Edit HTML/CSS visually inside VS Code — no browser needed.

![Zaemit Editor Overview](media/screenshots/ref1.png)

## Features

### Visual Drag & Drop Editing
Select, move, resize, and rotate elements directly on the canvas. What you see is what you get.

![Visual Editing](media/screenshots/ref3.png)

### Powerful Style Panel
Edit CSS properties with intuitive controls — layout, typography, borders, backgrounds, effects, and more. No need to write CSS by hand.

![Style Panel](media/screenshots/ref4.png)

### Responsive Design
Switch between Desktop, Tablet, and Mobile viewports instantly. Design for every screen size.

### Three Steps to Visual Perfection

![Getting Started](media/screenshots/ref2.png)

1. **Install the Extension** — Search "Zaemit" in VS Code Marketplace and install with one click
2. **Open Your Project** — Right-click any HTML file and select "Open with Zaemit"
3. **Design & Ship** — Edit visually, see code update live. Export production-ready code

## More Features

- **Layer Panel** — Navigate and reorder elements in the DOM tree
- **Undo / Redo** — Full change history with Ctrl+Z / Ctrl+Y
- **Code Editor** — Built-in code editor with syntax highlighting
- **Multi-select** — Select and edit multiple elements at once
- **Keyboard Shortcuts** — Efficient workflow with customizable shortcuts
- **Element Inspector** — View and edit element properties, attributes, and styles
- **Zoom & Pan** — Zoom in/out and pan across the canvas freely

## Usage

1. Right-click any `.html` file in the Explorer → **Open with Zaemit Visual Editor**
2. Or click the **Zaemit** button in the bottom status bar

## Requirements

- VS Code 1.85.0 or later
